이 문서는 현재 스크립트 설치법 전반을 포함합니다.
###설치 방법###

1. 모든 파일을 Zip으로 다운로드 합니다.
2. %APPDATA%/Roaming/McNeel/Rhinoceros/8.0/Scripts/Elephant
3. 파일 경로 안에 Elephant 폴더가 없다면 새로 생성해줍니다.
4. Elephant 폴더 안에 모든 파일은 압축해제하여 넣습니다.
5. ElephantTools.rhc 파일을 라이노 상에 드래그&드롭합니다.
6. 툴바에 보이는 #NEW# 버튼을 좌틀릭하여 명령어를 업데이트 합니다.
7. #NEW#버튼을 우클릭하면 추가된 명령어 리스트를 볼 수 있습니다.
